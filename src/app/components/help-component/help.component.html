Help 

    The site is composed of two components:
    - an API
    - a frontEnd
    
    Both are available on github : 
    genealogy-api
    genealogy-web
    
    Feel free to contribute.
    
    

    
    They are currently hosted on aws.
    
    a build is triggered using travis-ci 
    
    travis - genealogy-api
    travis - genealogy-web
    
    Once tests executed successfully, it is deployed manually
     Res01.com
Personal Data 
    By default, names and year of birth/death are visible to everyone.
    Other informations are only visible logged users.

    To register, you should find your profile first (if it exists) and then
    claim the profile.

    Roles will be implemented later to avoid sharing all the informations with
    everyone.
