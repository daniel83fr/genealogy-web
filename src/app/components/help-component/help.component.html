Technical Info 
    The website is hosted for free on Heroku.
    After 30 mins of inactivity, the server is sleeping, and takes a while to restart.
Personal Data 
    By default, names and year of birth/death are visible to everyone.
    Other informations are only visible to admins and profile owners.

    Roles:
    - Normal user:
      Can see simplified profile, suggest update  
    - Logged user:
      Can see full profile owned / releated persons
    - Admin:
      Full rights
